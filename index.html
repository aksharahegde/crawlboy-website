Basic Usage

Crawlboy provides a declarative API for defining your scraping logic. Import the crawler, define your target, and start the engine.

main.py 

from crawlboy import Crawler

# Initialize the kinetic engine
bot = Crawler(
    base_url="https://example.com",
    depth=2,
    concurrency=10
)

@bot.on_found
def handle_data(url, content):
    print(f"Discovered: {url}")

# Ignite the crawl
bot.ignite()
