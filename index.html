What's inside

Why Crawlboy?

A focused tool that does one thing well: turn a sitemap into a directory of Markdown files you can use for RAG, documentation mirrors, or offline reading.

travel_explore

Sitemap Discovery

Auto-detect the sitemap from robots.txt or common paths. Follows nested <sitemapindex> entries recursively.

$ --site-url https://example.com
article

Markdown Output

One .md file per page, mirroring the URL path. Optional raw HTML under html/ with --save-html.

$ --out-dir ./output
image

Image Download

Save images to media/, content-addressed and deduplicated. Enable with --download-images.

$ --download-images

1

URL paths become file paths

/blog/articles/basic-git-commands/ becomes out/md/blog/articles/basic-git-commands.md. The site root becomes index.md.

2

Optional HTML mirror

Pass --save-html to also write the raw HTML under html/, mirroring the same directory structure.

3

Content-addressed images

With --download-images, images are saved to media/ using content hashes вЂ” duplicates across pages are stored only once.

!

Failure tracking

Pages that fail to crawl are recorded in errors.jsonl. Use --fail-fast to stop on the first error instead.

Get started

Quick Start

install via pip
$ pip install crawlboy
$ crawl4ai-setup
Run crawl4ai-doctor to verify the browser setup is correct.
# Example usage:
$ crawlboy --sitemap-url 'https://example.com/sitemap.xml' --out-dir ./out
install from source
$ git clone https://github.com/aksharahegde/crawlboy
$ cd crawlboy
$ pip install -e .
$ crawl4ai-setup
run with docker
$ docker build -t crawlboy .
$ docker run --rm -v $(pwd)/out:/out crawlboy \
  --sitemap-url https://example.com/sitemap.xml \
  --out-dir /out
Docker runs headless (no display required) out of the box.
All flags

CLI Reference

Flag Description Default
--sitemap-url Direct sitemap URL to crawl вЂ”
--site-url Site root URL вЂ” sitemap is auto-discovered from robots.txt or common paths вЂ”
--out-dir Directory to write output files into вЂ”
--delay Seconds to wait between each page crawl 0
--page-timeout-ms Navigation timeout in milliseconds per page 60000
--max-urls Cap the number of URLs to crawl unlimited
--save-html Write raw HTML files under html/ in addition to Markdown off
--download-images Save images to media/, content-addressed and deduplicated off
--no-headless Show the browser window while crawling (disables headless mode) off
--fail-fast Stop the entire crawl on the first page error off
--include-offsite-urls Allow crawling URLs on hosts outside the origin site off
-i, --interactive Launch the guided wizard instead of using flags off
